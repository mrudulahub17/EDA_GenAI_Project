# 📊 EDA with GenAI

## 🚀 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** using Python and Generative AI concepts. The goal is to understand the dataset, identify missing values, analyze relationships between variables, and generate meaningful insights through data visualization.

---

## 🎯 Objectives

✅ Understand dataset structure and features

✅ Perform statistical analysis

✅ Detect missing values

✅ Analyze correlations between variables

✅ Generate visualizations

✅ Derive business insights using EDA techniques

---

## 📂 Dataset

The dataset contains the following features:

* 👤 Age
* 💰 Income
* 📈 Credit Score
* 🏦 Loan Amount
